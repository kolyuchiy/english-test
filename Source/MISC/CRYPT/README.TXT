
Вы можете изменить содержимое имеющегося теста или
создать свой. Для этого надо:
 * Отредактировать файл теста (с расширением tst) в
   любом текстовом редакторе (если в файле использу-
   ются русские буквы, то редактор должен уметь работать
   с русской dos-кодировкой cp866), соблюдая при этом
   следующие правила:
    - символом '#' в начале строки помечаются коментарии,
      которые не будут отображаться в процессе тестирования.
    - коментарий (строка с символом '#' в начале), находящийся
      на первой строке файла - название теста, которое будет
      отображаться в меню.
    - каждый вопрос состоит из двух частей: собственно
      вопрос и четыре варианта ответа.
    - ответы помечаются символами '??' в начале строки.
      Правильным является первый вариант ответа (в процессе
      тестирования варианты ответов перемешиваются случайным
      образом).
    - Количество вопросов неограничено.
 * Зашифровать файл теста с помощью программы encrypt.exe.
   При этом командная строка будет выглядеть примерно так:
      > encrypt file.tst file.enc
   где file.tst - незашифрованный файл,
       file.enc - зашифрованный файл.
   Шифрование применяется для того, чтобы тестируемый ученик
   не мог подсмотреть правильные ответы.
 * Скопировать исправленный файл в каталог тестов. Вы можете
   предварительно сохранить старый файл, если он вам понадобится
   в будущем.
